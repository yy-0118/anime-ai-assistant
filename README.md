# 🎭 动画AI助手

一个基于 **LangChain + OpenAI API** 的动漫风格 AI 助手，支持角色设定与上下文记忆。

## 🚀 快速开始

1. 克隆仓库
bash

git clone https://github.com/你的用户名/anime-ai-assistant.git

cd anime-ai-assistant

纯文本
2. 安装依赖
bash

pip install -r requirements.txt

纯文本
3. 配置环境
- 创建 `.env` 文件，填入你的 API Key
env

OPENAI_API_KEY=你的密钥

纯文本
4. 运行
bash

python main.py

纯文本
## 📦 技术栈
- Python 3.10
- LangChain
- OpenAI API
- python-dotenv

## ✨ 功能
- 动漫角色扮演
- 对话记忆功能
